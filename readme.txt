== Description ==
Display information of reserved posts, pages, and custom posts on the dashboard.


== Installation ==
1. Upload the 'reserve post board' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the Plugins menu in WordPress.

== Use filter hook ==
You can specify the following values by a filter hook.
1. Target post_type.
2. Time range until posts' publishing time in 'Publishing Soon' area.
3. Maximum display post count for future post list.
